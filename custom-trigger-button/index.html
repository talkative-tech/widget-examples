Talkative Chat Widget Example

This code example allows you to start an interaction using a custom call-to-action button which can be located anywhere within your page.

To make this example work with your own chat widget, please update the variables in the main.js file with the correct widget UUID and the correct actionable name.

The buttons below have a default text to indiciate the queue is offline. The is intentional as the JS will check the queue presences and then update the button text if the queue is online. If you don't want the button to appear you could instead set the styling for the button to be hidden by default, and then make the element visible once the queue check is complete.
