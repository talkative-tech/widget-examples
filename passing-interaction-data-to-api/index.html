Talkative Chat Widget Interaction Data via API

Talkative offers a pre-interaction data collection form to collect customer data. In some cases, the widget might appear on a part of a website where the user has already logged in. In this case, the pre-interaction form is not required, and the data can be passed to the widget via the API. If you still wish to use the pre-interaction form, you can still pass the data to the widget via the API, and it will be used to pre-populate the form.
